Tentang Saya

Halo saya Muhammad Rizky Bagas, seorang programmer freelancer dengan latar belakang pendidikan di SMK Telekomunikasi Tunas Harapan, dan saat ini saya sedang melanjutkan studi di Institut Teknologi Sains Bandung (ITSB) dengan jurusan Data Science.

Saya memiliki keahlian yang kuat dalam Web Development, khususnya dalam penggunaan framework PHP Laravel dan JavaScript seperti VueJS. Selain itu, saya juga berpengalaman dalam desain aplikasi menggunakan Figma, di mana saya berhasil meraih posisi 10 besar dalam kompetisi desain aplikasi di Universitas Diponegoro.

Web Development
